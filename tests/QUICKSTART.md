## Quick Verification (5 minutes)

### 1. Automated Code Verification

This runs quickly and verifies all code is in place:

```bash
# Windows
cd tests
run_diarization_tests.bat

# Unix/Mac
cd tests
chmod +x run_diarization_tests.sh
./run_diarization_tests.sh
```

This will:
- Run Python unit tests
- Run Rust unit tests
- Show manual testing checklist
