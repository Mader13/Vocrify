### 🚀 Как использовать в приложении

#### Вариант 1: Через интерфейс (GUI)

```bash
# Запустите приложение
bun run tauri:dev

# В приложении:
# 1. Перетащите файл 123.mp4 (или другой) в окно
# 2. Выберите модель (whisper-base для скорости)
# 3. Нажмите "Add to Queue"
# 4. Дождитесь завершения транскрипции
# 5. Экспортируйте результат в нужном формате
```

#### Вариант 2: Через командную строку

```bash
cd ai-engine

# Активируйте виртуальное окружение
venv\Scripts\activate

# Запустите транскрипцию
python main.py --file ../123.mp4 --model whisper-base --device cpu --cache-dir ./models

# С диаризацией (разделение по говорящим)
python main.py --file ../123.mp4 --model whisper-base --device cpu --diarize --cache-dir ./models
```
